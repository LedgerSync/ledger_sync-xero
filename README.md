# LedgerSync::Xero

[ledgersync.dev](www.ledgersync.dev)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ledger_sync-xero'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ledger_sync-xero
